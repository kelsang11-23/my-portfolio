# 个人自我介绍网站需求文档

## 1. 网站概述

### 1.1 网站名称

个人自我介绍网站

### 1.2 网站描述

展示个人综合素养和能力的个人网站，用于介绍自己的基本信息、教育背景、兴趣爱好和技能特长

## 2. 网站功能

### 2.1 个人信息展示

- 基本信息：姓名、年龄、联系方式、家乡等
- 教育背景：在读院校、专业等
- 兴趣爱好：个人兴趣、业余爱好、特长等
- 技能特长：专业技能、语言能力、计算机技能等

### 2.2 页面导航

- 首页：基本信息概览
- 关于我：详细个人介绍
- 联系方式：提供邮箱、电话等联系信息

## 3. 网站设计风格

### 3.1 配色方案

采用创意个性的马卡龙色系，主要色彩包括薄荷绿、樱花粉、天空蓝、薰衣草紫等柔和色调

### 3.2 视觉细节

- 圆角设计：大面积使用圆角增加亲和力
- 卡片式布局：内容采用卡片形式展示，层次分明
- 插画元素：融入手绘风格插画点缀页面
- 渐变背景：使用温和的渐变色彩营造丰富层次

### 3.3 整体布局

采用简洁的单页滚动布局，各个模块垂直排列，适合移动端和桌面端浏览,可以在github上部署
